## 这个项目是干什么的 

- 管理，保存我的环境和软件的配置

- 主要在 macos 上使用, 部分时间在 ubuntu 上使用

- 内部包含
  - kitty 配置文件
  - tmux 配置文件
  - 一个简化的 vim 配置文件
  - 一个 neovim 的启动器，用于启动 LichtVim 配置。
  - zsh 配置文件
  - 以上所有配置的安装及卸载脚本

## 文件结构

```
.
```
