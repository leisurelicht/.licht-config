# Vim 配置

---

- 安装/卸载

  - install.sh
  - uninstall.sh

- 文件结构

  - vimrc

    vim 入口文件

  - custom

    自定义配置

  - autoload

    vim 函数

  - installs

    要安装的第三方库

  - plugs

    第三方插件

  - config

    vim 配置

  - Vimscript

  - Go
